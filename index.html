approach

The overall approach is built off of 538's live win probability predictions. Starting there we see how the two team's probability of winning the game can fluctuate greatly throughout the contest. We are going to use how close these lines are to one another as a proxy for how close the game was.

This is an example of how animation can be leveraged in data visualization and data storytelling.
